# 🌟 Xeno's Personal Portfolio

A modern, responsive personal portfolio website showcasing web development projects, skills, and capabilities. Built with clean HTML5, CSS3, and vanilla JavaScript.

## 📋 Features

✨ **Modern Design**
- Sleek, dark theme with gradient accents
- Responsive layout (mobile-first approach)
- Smooth animations and transitions
- Professional typography with Google Fonts

🎯 **Interactive Elements**
- Mobile-responsive navigation menu
- Scroll-triggered navigation highlighting
- Contact form with mailto functionality
- Interactive project cards with hover effects

📱 **Fully Responsive**
- Works perfectly on desktop, tablet, and mobile devices
- Tested on all modern browsers
- Optimized performance

## 🚀 Quick Start

### View Locally
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate using the menu or scroll

### Using Python HTTP Server (Testing)
```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`

## 📧 Contact Form

The contact form uses **mailto** functionality:
- Clicking "Send Message" opens your default email client
- Pre-fills your email and message content
- No backend required

To modify the recipient email, edit the `script.js` file and update the email address in the contact form handler.

## 🌐 Deployment

### Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com)
2. Sign up and create a new site
3. Drag and drop this folder
4. Your site goes live instantly!

### GitHub Pages
1. Push this repository to GitHub
2. Enable GitHub Pages in repository settings
3. Set source to `main` branch
4. Site will be live at `https://yourusername.github.io/portfolio`

### Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Deploy with one click!

## 🐛 Troubleshooting

**Links not working?**
- Ensure you're using the correct filename (case-sensitive on Linux/macOS)
- Check that all files are in the same directory

**Styles not loading?**
- Make sure `styles.css` and `love.css` are in the root folder
- Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)

**Forms not working?**
- Check your default email client is set up
- Browser may prompt for permission to open email client
